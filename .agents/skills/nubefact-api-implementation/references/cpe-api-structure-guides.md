# CPE API — PARA GUÍAS

> Source: `assets/cpe-manual-google-doc.md` — PARA GUÍAS

| PARA GUÍAS |  |  |  |  |
| :---- | :---- | :---- | :---- | :---- |
| ATRIBUTO | DESCRIPCIÓN | TIPO DE DATO | REQUISITO | LONGITUD |
| guia\_tipo | Tipo de guía relacionada. 1 = GUÍA DE REMISIÓN REMITENTE, 2 = GUÍA DE REMISIÓN TRANSPORTISTA | Integer | Opcional | 1 exacto |
| guia\_serie\_numero | Serie y número de guía relacionados separados por guión. Ejemplo: 0322-1 | String | Opcional | 1 hasta 30 |
