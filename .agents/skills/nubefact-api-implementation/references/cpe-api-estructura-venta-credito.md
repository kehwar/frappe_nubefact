| PARA VENTA AL CRÉDITO |  |  |  |  |
| :---- | :---- | :---- | :---- | :---- |
| ATRIBUTO | DESCRIPCIÓN | TIPO DE DATO | REQUISITO | LONGITUD |
| cuota | Número de cuota. Ejemplo: 1 | Integer | Opcional | 1 hasta 3 |
